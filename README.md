1. 填写URL和How many object*信息
2. 按init
3. 按run once 来运行一次攻击
或者:
1. 完成所有输入框
2. 按do loop 来运行无限的攻击
停止循环:
* 刷新(当你感到严重卡顿时,这是最好的方法,只不过会丢失表单信息)
* 按cancel loop按钮
